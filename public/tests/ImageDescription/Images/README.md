# Image Description Test - Images

Place 5 images in this directory with the following names:
- image1.jpg
- image2.jpg
- image3.jpg
- image4.jpg
- image5.jpg

These images will be randomly selected for the image description test.

## Recommended Image Types:
- Complex scenes with multiple objects
- Everyday scenarios (e.g., park, kitchen, street scene)
- Images with people performing activities
- Nature scenes with various elements
- Indoor/outdoor settings with details

## Image Requirements:
- Format: JPG, PNG
- Resolution: At least 800x600 pixels
- Clear and well-lit
- Appropriate content for cognitive assessment
