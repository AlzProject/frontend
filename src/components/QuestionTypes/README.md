# Question Types Components

This directory contains reusable components for building cognitive and mental health assessments.

For detailed usage instructions and examples, please refer to [COMPONENTS_USAGE.md](../../COMPONENTS_USAGE.md) in the `src` root.

## Quick Reference

- **`MultipleChoiceQuestion`**: Radio buttons (single) or Checkboxes (multi).
- **`LikertScaleQuestion`**: Rating scales (1-5, etc.).
- **`TextResponseQuestion`**: Text inputs (single line) or Textareas (multi-line).
- **`DrawingCanvasQuestion`**: Canvas for drawing tasks (Clock Drawing, Figure Copying).
- **`QuestionWrapper`**: Container with title and styling.
